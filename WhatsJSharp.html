What's J#?

J# is a programming language that combines elements of Javascript and C#. It is designed to be simple and easy to use, making it accessible for beginners while still powerful enough for advanced users.

Features (and coming ones)

  1. Beginner friendly, very simple objects, and functions.
  2. Notices error then shows it when the code is ran.
  3. Supports coloring, math, and class functions (Coming soon!).

J# is a versatile language that can be used for a variety of applications, from web development to game development. It is designed to be easy to learn and use, making it a great choice for beginners and experienced programmers alike.

J# is still in development, and new features are being added regularly. If you have any suggestions or feedback, please feel free to reach out.
